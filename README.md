# GGMUX
Alat untuk mempercantik tampilan termux dengan mudah tanpa ribet edit script

<h3>Installing</h3>

```java
$ apt update && apt upgrade
$ apt install git
$ apt install cowsay
$ apt install ruby
$ apt install figlet
$ gem install figlet
$ git clone https://github.com/ArroKM/GGMUX
$ cd GGMUX
$ bash ggmux.sh
```
